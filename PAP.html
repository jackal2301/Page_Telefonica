PAP (Paso a Producción)

Actividades previas al PAP

  • Al iniciar un PAP se debe establecer una fecha de ejecucion y el analista debe confirmar ingresando a una plataforma establecida para su respectiva aprovacion.
  • Se debe tener muy en claro el alcance de las historias de usuario y funcionalidades a desplegar para que el PAP sea mas agil pero efectivo.
  • Se debe tener una de suite de pruebas con los casos correspondientes para que el aanalista sea centrado a lo que se tiene que realizar.
  • Es muy importante que el analista cuente con los insumos suficientes para desplegar el PAP para que en el proceso no se retrace.
  • Tener una VPN establecida es escencial para el despliegue por que asi el analista cuenta con los accesos necesarios para el PAP.
  • El analista debe tener en cuenta las precaucuones que se deben tener en caso que algo salga mal durante el despliegue, se debe informar al scrum encargado del tipo de falla o error generado.
  • El analista debe tener presente que cuenta con un equipo de soporte y debe saber cual es su equipo de soporte para el despliegue.

Durante el PAP

  • Al analista se le debe confirmar el momento exacto para iniciar pruebas del despliegue para tener una coordinacion con el equipo de trabajo
  • El analista debe tender presente si en el despliegue se realizaran pruebas automatizadas para contar con las herramientas suficientes para probar
  • Verificar al finalizar el PAP que las pruebas de continuidad fueron realizadas con exito
  • Para culminar el despliegue correctamente el analista debe generar las evidencias de todas las pruebas realizadas para tener soporte de casos exitosos y casos fallidos

Posterior al PAP

  • El analista debe chequear que se finalizaron todas las validaciones en el despliegue
  • Para dar por finalizado el despliegue el analista debe enviar correo con las evidencias correspondidentes y el documento de certificacion para aprobar el PAP.

Beneficios del PAP realizado desde la empresa

El analista cuando realiza el paso a producccion desde las instalaciones de la empresa tiene derecho a una bonificacion para pedir comida despues de las 8 pm y si el PAP se se vuelve mas extenso el analista tiene derecho a una bonificacion de transporte velando por la seguridad del mismo

Beneficios del PAP realizado desde casa

El analista cuando realiza un paso de produccion desde casa cuanta con el beneficio de que si el PAP se vuelve extenso y se termina en horas de la madrugada el analista puede quedarse en casa a realizar homework
