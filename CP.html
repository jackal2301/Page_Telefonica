CASOS DE PRUEBA

Es un conjunto de condiciones o variables bajo las cuales un analista determinará si una aplicación, un sistema de software (software system), o una característica de éstos es parcial o completamente satisfactoria. Es decir, un caso de prueba es un conjunto de pasos y resultados esperados que se crean a partir de los requisitos del software que se va a probar.
